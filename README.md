## About

A Unity 2D top-down controller with 8-directional movement and persistent directional idle animations.

## Features

- 8-directional Rigidbody2D movement
- Directional idle animations with last movement direction memory
- Dust particle effects
- Pivot-based sprite sorting

## How to Run

1. Open the project using Unity 6000.4.5f1 or newer
2. Open the `Assets/Scenes/SampleScene.unity` scene
3. Press Play
4. Use WASD or Arrow Keys to move the character

## Assets

Character sprites are based on the [8-Directional Game Boy Character Template by GibbonGL](https://gibbongl.itch.io/8-directional-gameboy-character-template) and were modified/redrawn for this project.
